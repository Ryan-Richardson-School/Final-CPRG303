## Project Description
Orange Minds is a mobile trivia application designed to run on Android devices. The application allows users to select a trivia category and answer multiple choice questions. The app provides immediate feedback after each answer and tracks the user's score throughout the quiz.

## Features
- Multiple trivia categories
- Randomly generated quiz questions
- Score tracking
- User profile and account settings
- Login functionality
- Vibration feedback for correct and incorrect answers
